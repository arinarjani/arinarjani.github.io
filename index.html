Everything below works in this window using <canvas> or <iframe>

Page( 2/3 ) This is the Color Guessing game I created from the Web Developer Bootcamp. When you reload the page, the top RGB color changes to a randomly generated color chosen. New colors also generate when the user clicks: "new colors", "easy", or "hard". You have to click on each square until you guess the correct color. All this was done with JavaScript.

Page( 2/3 ) This is the To-Do list app I created from the community page of Gomix under the Tools for Work. The original is done by "enriquesanchez" and can be seen here: Original Todo. Instructions include pressing "Enter" or clicking the "+" sign. To delete items simply click/touch on them. I still need to learn how to use localstorage, but I learned some new techniques with respect to scope is JS.

Page( 2/3 ) This is the Patatap game clone I created from the Web Developer Bootcamp. The user can press keys A-Z to create randomly placed circles that animate away. Also a soound plays when a key is pressed that is specific to each A-Z letter. The background color can be changed when the user presses the enter key. This was done with Paper JS ( animation ), Howler JS ( audio ), and Neuronal-Synchrony ( sounds ). For now this only works with a keyboard; touch soon.
