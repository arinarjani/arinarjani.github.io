I am still working on a portfolio site, but below is my resume.

This is the Color Guessing game I created from the Web Developer Bootcamp. When you reload the page, the top RGB color changes to a randomly generated color chosen. New colors also generate when the user clicks: "new colors", "easy", or "hard". You have to click on each square until you guess the correct color. All this was done with JavaScript.

This is the To-Do list app I created from the Web Developer Bootcamp. You can hide/show the input field by clicking the "x" or "+" sign. When you enter a new to-do item, it is added to the page. Also when you hover over an item, the red trashcan appears to allow the deletion of the task. All this was done with JQuery.

This is the Patatap game clone I created from the Web Developer Bootcamp. The user can press keys A-Z to create randomly placed circles that animate away. Also a soound plays when a key is pressed that is specific to each A-Z letter. The background color can be changed when the user presses the enter key. This was done with Paper JS ( animation ), Howler JS ( audio ), and Neuronal-Synchrony ( sounds ).
