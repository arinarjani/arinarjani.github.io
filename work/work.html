RGB Color Guessing Game

Design & Development

This is the Color Guessing game I created from The Web Developer Bootcamp. The object of the game is to guess the color at the top of the page. When you reload the page, the top RGB color changes to a randomly generated color chosen. New colors also generate when the user clicks: "new colors", "easy", or "hard". You have to click on each square until you guess the correct color. All this was done with JavaScript. The course on Udemy.
